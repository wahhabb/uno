# Online Uno Game
You are going to build a project that will run in web browsers. It allows two or more people in different locations to play the game Uno together. There may be multiple groups playing independently of each other. 
The game will display on the screen with animations and the sound of a card flip with cards are dealt or moved.
## Tech Stack
* The site will be written using SvelteKit.
* Use native CSS
* Use TypeScript
* The database will use Supabase
* The Uno card images with be SVGs for efficiency
